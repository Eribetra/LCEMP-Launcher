03/02/2026
Hello! This is a launcher for Windows builds of Minecraft: Legacy Console Edition.

Features:
- Starting a Windows LCE build from a chosen folder (must have Minecraft.Client.exe present)
- Setting a custom username and IP address (for builds with multiplayer support)
- Saving and loading all 3 parameters.
